# 🏨 Hotel Booking System

A full-stack **Hotel Booking Web Application** built using **Node.js, Express, MongoDB, and JavaScript**.
This project allows users to book rooms, calculate prices, and contact the hotel, while providing separate admin panels for managing bookings and messages.

---

## 🚀 Live Demo

🔗 https://hotel-booking-system-8xru.onrender.com

---

## 📌 Features

### 👤 User Side

* 🔐 User Registration & Login
* 🏨 Room Selection with **“Book Now” navigation**
* 📅 Booking system with check-in & check-out
* 💰 Dynamic price calculation
* 💳 “Pay Now” simulation
* 📩 Contact form

---

### 👨‍💻 Admin Side

* 📊 View all bookings (`admin-bookings.html`)
* 📩 View all contact messages (`admin-contacts.html`)

---

## 🛠️ Tech Stack

* **Frontend:** HTML, CSS, JavaScript
* **Backend:** Node.js, Express.js
* **Database:** MongoDB

## ⚙️ Installation & Setup

### 1️⃣ Clone the repository

```
git clone https://github.com/sablekalpesh07-ui/hotel-booking-system.git
cd hotel-booking-system
```

### 2️⃣ Install dependencies

```
npm install
```

### 3️⃣ Create `.env` file

```
MONGO_URI=your_mongodb_connection_string
PORT=5000
```

### 4️⃣ Run the server

```
node server.js
```
